# Douby

## Brief introduction
Douby is the application which follows Material Design and filters movies by rating.
What's more, the results by analyzing emotions from comments is provided.

## Open source project want to practice
+ `RxJava`    
+ `Retrofit`  
+ `OkHttp`    
+ `Fresco`  
+ `Glide`
+ `Realm`
